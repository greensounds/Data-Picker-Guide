Gráfico de pie

Definición:

Los gráficos de pie muestran los datos como una proporción del total. Los gráficos pie se usan normalmente para realizar comparaciones entre grupos. Los gráficos de pie, junto con los gráficos piramidales y de embudo, forman parte de un grupo de gráficos conocidos como gráficos de formas. Los gráficos de formas no tienen ejes. Cuando se coloca un campo numérico en un gráfico de formas, el gráfico calcula el porcentaje de cada valor en relación con el total.

Herramientas para generar la visualización

Mediante una interfaz (Sin usar de código)

  • HighCharts Cloud
  • AM Charts Live
  • Live Gap Charts

Usando código JavaScript

  • D3.js
  • AnyChart.js
  • Chart.js
