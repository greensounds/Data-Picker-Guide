Selección de gráficos

Criterios recomendados para la elección de un gráfico

Los gráficos nos ayudan a visualizar información de manera fácil y sencilla, sin embargo uno de sus principales problemas es elegir el gráfico adecuado. Tenemos todo tipo de gráficos desde: gráficos de barras, de columnas, de pie, histogramas, de dispersión, de burbujas, mapas geográficos, etc.

Después de analizar tres propuestas de diferentes autores (Andrew Abela, Severino Ribbeca, y Ann K. Emery). Se propuso la utilización de cuatro principales criterios para la selección de gráficos, los cuales son: comparación, relación, composición y distribución. Dentro de cada criterio se encuentran los gráficos que se recomienda utilizar. Algunos gráficos se pueden utilizar para mas de un criterio.
