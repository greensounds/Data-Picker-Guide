Data Picker Guide

Acerca del sitio web

El sitio "Data Picker Guide" es un proyecto desarrollado por Jesús Díaz como parte de una investigación de la Maestría en Comunicación con Medios Virtuales del Insituto de Investigación en Comunicación y Cultura ICONOS

El sitio pretende servir como una guía para diseñadores y desarrolladores web que quieren crear visualizaciones de datos pero tropiezan con algunas dudas, por ejemplo, que herramientas existen para realizar gráficos, o que tipos de gráficos son los adecuados dependiendo de lo que se quiera transmitir, o que paleta de colores o tipografías podemos utilizar para sacar el mayor provecho de las visualizaciones de datos.

El sitio web está dividido en cinco secciones: gráficos, colores, tipografía, librerías y recursos. Dentro de la primera sección se encuentran los criterios que se proponen para la elección de un tipo de gráfico. También se encuentran los diferentes tipos de gráficos que se proponen en cada criterio, además de una pequeña definición y un pequeño ejemplo del gráfico con el objetivo de que el usuario se familiarice con los gráficos

En la sección de "Colores" encontrarás algunas paletas de colores que se porponen para la visualización de datos, además de las características que hay que tomar en cuenta al elegir alguna paleta de colores.

En la sección de "Tipografía" encontrarás las tipografías que se recomiendan para la visualización de datos así como las características que se recomienda tomar en cuenta al momento de realizarlas.

En la sección "Librerías" encontrarás seis librerías JavaScript que fueron analizadas para esta investigación. Las librerías están divididas en dos secciones y contienen tres librerías cada una. La primera sección se enfoca a las librerías que nos permiten crear visualizaciones de datos sin la necesidad de utlizar código de programación, es decir, las visualizaciones son realizadas mediante una interfaz web y al finalizar podemos exportar el código generado y solo copiar y pegar donde se vayan a utilizar. La segunda sección se enfoca en las librerías que son realizdas solamente mediante código (JavaScript en este caso).
