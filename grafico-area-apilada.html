Gráfico de área apilada

Definición:

Los gráficos de área apilada, a diferencia de los gráficos de línea representan visualmente el volumen. La información se coloca en dos ejes y se conecta mediante puntos. El área entre los ejes y los puntos se enfatiza con algún color. Los gráficos de área regularmente se utilizan para comparar dos o mas categorás

Herramientas para generar la visualización

Mediante una interfaz (Sin usar de código)

  • HighCharts Cloud
  • AM Charts Live
  • Live Gap Charts

Usando código JavaScript

  • D3.js
  • AnyChart.js
  • Chart.js
