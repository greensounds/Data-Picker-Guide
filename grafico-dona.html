Gráfico de dona

Definición:

Un gráfico de dona muestra datos como porcentajes como parte de un todo. Igual que en los gráficos de pie, las categorás se representan mediante una porción dentro del gráfico.

Herramientas para generar la visualización

Mediante una interfaz (Sin usar de código)

  • HighCharts Cloud
  • AM Charts Live
  • Live Gap Charts

Usando código JavaScript

  • D3.js
  • AnyChart.js
  • Chart.js
