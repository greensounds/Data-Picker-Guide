Mapa de Conexiones

Definición:

Los mapas de conexiones se realizan al conectar dos puntos mediante líneas rectas o curvas.Estos mapas se utilizan para mostrar relación geográficas o para dibujar rutas.

Herramientas para generar la visualización

Mediante una interfaz (Sin usar de código)

  • HighCharts Cloud
  • AM Charts Live
  • Live Gap Charts

Usando código JavaScript

  • D3.js
  • AnyChart.js
  • Chart.js
