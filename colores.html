Paleta de Colores

“El color usado correctamente puede mejorar y clarificar una presentación. El color usado pobremente causará confusión”
Maureen Stone.

Elección de paleta de colores para la visualización de datos

La paleta de colores es un punto fundamental para la realización de las visualizaciones de datos. Si se elige incorrectamente la paleta de colores puede que no se transmita de manera efectiva la información que esperamos comunicar.

Después de analizar a tres autores que hablan sobre elección de paletas de colores para la visualización de datos (Samantha Zhang, Andy Kirk, y Amy cesal)), se propone esta serie de recomendaciones para seleccionar una paleta de colores.

Recomendaciones

Utilizar Gradientes de color

Las gradientes de color son la mejor opción para trabajar una visualización de datos. Dicha gradiente es recomendable que que tenga un rango amplio de brillo, contraste y matiz.

Ejemplos de paletas de colores propuestos por Samantha Zhang

Accesibilidad

Un punto que no se suele tomar en cuenta al momento de crear visualizaciones de datos es la accesibilidad para la personas que son daltónicas. Se recomienda que al momento de seleccionar alguna paleta de color utilizamos alguna herramienta como Vizcheck. Esta herramienta nos permite visualizar nuestra paleta y verla como la vería un daltónico. Es una buena medida que mejorará nuestra visualización y será accesible a una mayor cantidad de personas.
