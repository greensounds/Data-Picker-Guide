Librerías JavaScript para la visualización de datos

En esta sección se encuentran sintetizada la investigación que se realizó de seis librerías JavaScript para la visualización de datos. Las librerías se clasifican en dos grupos, tres de ellas fueron analiazadas pensando en un perfil de diseñador web y las otras tres fueron pensando en un perfil de desarrollador web.

Las librerías analizadas pensando en un perfil de diseñador tienen como característica principal que se realizan mediante una interfaz. Las librerías que fueron analizadas son: HighCharts, AM Charts y Live Gap Charts

Las librerías analizadas pensando en un perfil de desarrollador tienen como característica principal que se realizan mediante el uso de código JavaScript. Las librerías que fueron analizadas son: D3.js, AM Charts y Chart.js
