Mapa Coroplético

Definición:

Los mapas coropléticos o mapas de coropletas son una forma de cartografiado cuantitativo utilizada para la representación de fenómenos discretos asociados a unidades de enumeración (países, provincias o municipios) a las que se aplican símbolos superficiales de acuerdo con su valor. Los mapas coropléticos utilizan datos zonales y diferentes tonos de color o tramas para representar la distribución y densidad territorial de los valores de una escala.

Herramientas para generar la visualización

Mediante una interfaz (Sin usar de código)

Usando código JavaScript
